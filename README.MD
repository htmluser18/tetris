<center># Terminal Tetris (Python)</center>

A simple *Tetris game that runs entirely in the terminal*, built using Python and the standard curses library.  
This project is intentionally minimal so it can be *understood, extended, and explained easily* (especially for first-year projects).

## Controls

| Key | Action |
|----|-------|
| A | Move Left |
| D | Move Right |
| S | Move Down |
| W | Rotate Block |
| Q | Quit Game |

Controls are displayed *inside the terminal during gameplay*.


## How to Run

clone the code - git clone <url> 
run the code in python 3.13.7 or in code editor
